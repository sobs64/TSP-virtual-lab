Travelling Salesman Problem

Step 1: Add Cities

  • Click on the canvas to add cities (nodes) for the salesman to visit.
  • You can also import coordinates or distance matrix depending on the mode.

Step 2: Select Algorithm

  • Choose the algorithm to solve TSP: Brute Force, Nearest Neighbor, Genetic Algorithm, or others.

Step 3: Start the Simulation

  • Click on "Start" or "Solve" to begin the route calculation process.
  • The algorithm will compute the shortest path and visualize it on the canvas.

Step 4: Analyze Output

  • The optimal or best-found path is shown along with the total distance covered.
  • Time taken by the algorithm to compute is also displayed.

Step 5: Modify or Reset

  • Use the controls to clear cities, reset simulation, or change parameters and try different strategies.
